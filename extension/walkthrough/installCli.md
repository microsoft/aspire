# Install the Aspire CLI

The Aspire CLI is a cross-platform tool that helps you create, run, and manage Aspire applications.

You can install it with a package manager (such as Homebrew, winget, or apt) or with the install
script for your platform. The installation guide walks through every option and how to verify the
install with `aspire --version`.

> [Open the installation guide](https://aspire.dev/get-started/install-cli/)
